#### Installation

1.  pip3 install -r requirements.txt
2.  yun install -r nginx httpd

#### Instructions

1.  Build and configure nginx+uwsgi service
2.  Start uwsgi : uwsgi --ini uwsgi.ini
3.  Restart nginx : systemctl restart nginx.service
